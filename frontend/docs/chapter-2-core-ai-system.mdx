# Core AI Systems in Humanoid Robots

This chapter explains the essential AI technologies that enable humanoid robots to think, learn, and adapt.

## Learning Objectives
- Understand core AI models used in robots
- Explain perception, planning, and decision-making systems
- Identify learning techniques used in robotics

## Perception Systems
Robots use cameras, LiDAR, and microphones to interpret their surroundings through:
- Computer vision
- Object detection
- Depth estimation

## Decision-Making Systems
Robots rely on:
- Reinforcement learning
- Rule-based logic
- Motion planning algorithms

## Summary
Humanoid robots use integrated AI systems to perceive, plan, and act.
